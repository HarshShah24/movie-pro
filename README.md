Movie-Pro
Movie-Pro is a sleek, responsive web application designed for discovering and exploring the latest cinema. It features a modern dark-mode interface with high-quality posters, real-time search, and up-to-date ratings for upcoming and trending films.

🚀 Features
Live Search: Instantly filter through a vast database of movies.

Dynamic Ratings: Visual star ratings for every title (e.g., Predator: Badlands at ⭐ 7.7).

Responsive Grid: A clean, card-based layout that adapts to any screen size.

Modern UI: High-contrast dark theme for a premium cinematic feel.

🛠️ Tech Stack
Frontend: React.js / Vite

Data Source: The Movie Database (TMDB) API

Styling: CSS3 / Tailwind CSS

Icons: FontAwesome or similar SVG sets

Installation
Clone the repository:

Bash
git clone https://github.com/HarshShah24/movie-pro.git
Navigate to the directory:

Bash
cd movie-pro
Install dependencies:

Bash
npm install
Run the development server:

Bash
npm run dev
